## 3. 次バージョンの種別推奨

### 推奨: マイナーバージョン 15.1

**対象**: PHASE3 項目1.2「検索設定」（Web検索 ON/OFF・検索深度・取得件数の動的切り替え）

**理由**:

1. **既存基盤の拡張**: ver15.0 で構築した動的設定システム（`ThreadSettings` 型 + `settings` カラム + 設定 API + `SettingsPanel.vue` + `useSettings` composable）をそのまま拡張する。新しいアーキテクチャの導入は不要
2. **同一 MASTER_PLAN 項目の継続**: PHASE3 項目1「動的設定システム」のサブ項目1.2 であり、項目1.1 の完了後に自然に続くスコープ
3. **変更規模の見積もり**: `ThreadSettings` 型にフィールド追加 + `performSearch()` の設定参照 + `SettingsPanel.vue` の UI 拡張で、中規模の変更（推定 200〜400行）

### 15.1 の概要

- `ThreadSettings` に `search` フィールド（`{ enabled: boolean, depth: 'basic' | 'advanced', maxResults: number }`）を追加
- `performSearch()` が設定値を参照して Tavily パラメータを動的に変更
- `SettingsPanel.vue` に検索設定 UI（トグル・セレクト・スライダー）を追加
- 検索 OFF 時は `caseSearchNode` が LLM 内部知識のみで事例生成
