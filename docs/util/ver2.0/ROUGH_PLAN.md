# ROUGH_PLAN: util ver2.0

## 対応範囲

MASTER_PLAN PHASE2.0 の一部に着手する（項目 1・2・5 のログ関連部分）。

## 概要

`claude_loop.py` にワークフローログの永続化機能を追加する。

現状、ワークフロー実行時の出力はコンソールに流れるだけで、後から実行内容を確認する手段がない。本バージョンでは以下を実現する:

1. **ログファイルへの出力記録**: 各ステップの stdout/stderr をファイルに保存しつつ、従来通り端末にもリアルタイム表示する。ログにはステップごとの開始/終了時刻・所要時間・終了コードも記録する
2. **コミット記録**: ワークフロー開始前・各ステップ完了後の HEAD コミットハッシュをログに記録し、どのステップでどのコミットが生まれたかを追跡可能にする
3. **エージェントへのログパス共有**: ログファイルのパスを各ステップの Claude CLI に伝えることで、エージェントが前ステップの出力をログから読み取れるようにする。これによりエラー発生時の原因調査や、前ステップの結果を踏まえた判断が可能になる
4. **CLI オプション**: ログ出力先の指定やログ無効化など、ログ機能を制御するオプションを追加する

## スコープ外（後続バージョンで対応）

- ワークフロー完了通知（PHASE2.0 項目 3）
- 自動実行モードの設定ファイル化（PHASE2.0 項目 4）
- ログのローテーション・自動削除
