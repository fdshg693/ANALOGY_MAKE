## 機能概要

### 1. ワークフロー完了通知

ワークフロー完了（成功・失敗とも）時にデスクトップ通知を送り、バックグラウンド実行中でも完了に気付けるようにする。

- Windows トースト通知（PowerShell 経由）を第一候補、ビープ音をフォールバック
- `--no-notify` フラグで通知を無効化可能
- 通知内容: ワークフロー名、成功/失敗、所要時間

### 2. 自動実行モードの設定ファイル化

現在 `claude_loop.yaml` の `command.args` にハードコードされている自動実行モード設定を、YAML 上の `mode` セクションとして構造化し、CLI オプションで切り替え可能にする。

- `mode.auto: true/false` で自動/対話モードを定義
- `command.auto_args` に自動モード専用の引数を分離
- `--auto` / `--interactive` CLI オプションで YAML 設定を上書き可能
- 各エージェントに実行モードを `--append-system-prompt` で伝達
