# ROUGH_PLAN: util ver2.2

## 対応内容

MASTER_PLAN PHASE2.0 の残タスク（項目1の一部）を完了させる。

### 対応する項目

**未コミット変更の検出・警告 + `--auto-commit-before` フラグ**（PHASE2.0 項目1 残り）

ワークフロー開始前に `git status --porcelain` で未コミットの変更を検出し、ユーザーに警告する機能を追加する。また、`--auto-commit-before` フラグにより、開始前に自動コミットを行うオプションを提供する。

### 提供される機能

1. **未コミット変更の検出**: ワークフロー開始時に作業ディレクトリの状態をチェックし、未コミット変更がある場合は警告を出力する
2. **`--auto-commit-before` CLIフラグ**: 明示的に指定された場合のみ、ワークフロー開始前に未コミット変更を自動コミットする
3. **ログへの記録**: 未コミット変更チェックの結果をワークフローログに記録する

### 対応しないこと

- ワークフロー終了時の未コミット変更チェック
- 自動コミットのメッセージカスタマイズ
- `.gitignore` されていないファイルの個別警告

### バージョン種別

マイナーバージョン（ver2.2）。PHASE2.0 の残タスク完了であり、アーキテクチャ変更・新ライブラリ導入なし。
