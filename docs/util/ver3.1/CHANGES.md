## 変更内容の詳細

### `scripts/claude_loop.py` — コマンドログ出力の修正

**問題**: `_run_steps()` 内の `command_str = " ".join(command)`（489行目付近）が、スペースや特殊文字を含む引数をクォートせずに結合していた。これにより、`--append-system-prompt` の長文値と後続引数の境界がログ上で判別できなかった。

**修正**: `" ".join(command)` → `shlex.join(command)` に変更。`shlex.join()` はスペースや特殊文字を含む引数を自動的にシェルクォートするため、ログ出力でコマンド構造が明確になる。`shlex` は既にインポート済みだったため、追加のインポートは不要。

## 技術的判断

- `shlex.join()` は Python 3.8+ の標準ライブラリ関数。プロジェクトの Python バージョン要件を満たしており、外部依存の追加なし。
- テストファイル内の `" ".join` はトースト通知テスト用で今回の修正対象外。変更不要と判断。
