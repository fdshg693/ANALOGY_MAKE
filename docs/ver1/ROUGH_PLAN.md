# Task 1: チャットUI + モックAPIの構築

## 概要

アナロジー思考AIアシスタントの**基盤となるチャット画面とAPIルート**を構築する。
この段階ではAI連携は行わず、UIとサーバー間の通信パイプラインを確立することに集中する。

## 提供する機能

### チャット画面（1画面）

- メッセージ入力欄とチャット履歴の表示エリアを持つシンプルなチャットUI
- ユーザーのメッセージ送信（Enter / 送信ボタン）
- ユーザーメッセージとAIメッセージの区別表示
- 送信中の状態表示（ローディング）
- 自動スクロール（最新メッセージが常に見える）

### APIルート

- `POST /api/chat` エンドポイント
- リクエスト: ユーザーの会話履歴を受け取る
- レスポンス: モック応答を返す（固定テキストまたはエコー）
- 後続タスクでLangChain連携に差し替え可能な構造にする

### 画面イメージ

```
┌──────────────────────────────────┐
│  Analogy AI                      │
├──────────────────────────────────┤
│                                  │
│  [User] こんにちは               │
│                                  │
│  [AI]   モック応答: こんにちは   │
│                                  │
├──────────────────────────────────┤
│  [メッセージを入力...]    [送信] │
└──────────────────────────────────┘
```

## 技術スタック

- **フロントエンド**: Nuxt 4 + Vue 3（Composition API / `<script setup>`）
- **バックエンド**: Nuxt Server API Route（`server/api/chat.post.ts`）
- **スタイリング**: スコープドCSS（外部ライブラリなし）

## 完了条件

- ブラウザでチャット画面が表示される
- メッセージを入力して送信すると、モックAPIから応答が返りチャット履歴に表示される
- ユーザーとAIのメッセージが視覚的に区別できる
- 送信中はローディング状態が表示される
