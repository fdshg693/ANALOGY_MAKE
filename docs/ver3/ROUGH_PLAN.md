# ver3 タスク概要: サーバー API への LangChain 統合

## 概要

ver2 の実験で動作確認済みの LangChain + MemorySaver + アナロジー思考プロンプトを、Nuxt サーバー API (`/api/chat`) に統合する。
モック応答を廃止し、実際の OpenAI API を通じたアナロジー思考 AI アシスタントとしてチャットが機能する状態にする。

## 提供される機能

### 1. AI によるアナロジー思考対話

ユーザーが課題を入力すると、AI が以下の 5 ステップで対話を進める：

1. 課題の受け取りと確認
2. 課題の抽象化（分野に依存しない構造的な問題として再定義）
3. 他分野の類似事例を 3〜5 個提示（ステップ 2 と同一応答）
4. ユーザーが事例を選択
5. 選択された事例に基づく具体的な解決策の提案

### 2. 会話メモリの保持

MemorySaver を使用し、同一セッション内の会話履歴をサーバー側で自動管理する。
ユーザーは追加質問ややり直しを自然に行える。

## スコープ外（本タスクでは対応しない）

- CLI からの実行
- エラー時の UI フィードバック改善
- 会話履歴の永続化（DB 保存）
- 複数セッションの管理 UI
